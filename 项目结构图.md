## 三、成绩分析系统业务流程图

```mermaid
flowchart TD
    start([开始]) --> roleChoice{用户角色}
    
    roleChoice -->|教师| teacherInput[录入学生成绩]
    roleChoice -->|管理员| adminInput[批量导入成绩]
    roleChoice -->|学生| studentView[查看个人成绩]
    
    teacherInput --> validateInput[数据格式校验]
    adminInput --> validateInput
    
    validateInput --> checkValidation{校验通过?}
    checkValidation -->|否| validationError[返回错误提示]
    checkValidation -->|是| storeData[(存储到student_grades表)]
    
    storeData --> triggerAnalysis[触发成绩分析任务]
    triggerAnalysis --> queryTeacherCourses[查询教师授课记录]
    
    queryTeacherCourses --> checkRecords{记录存在?}
    checkRecords -->|否| generateErrorReport[生成错误报告]
    checkRecords -->|是| calculateMetrics[计算绩效指标]
    
    calculateMetrics --> performAnalysis[执行统计分析]
    performAnalysis --> generateReport[生成分析报告]
    generateReport --> storeReport[(存储分析结果)]
    
    storeReport --> notifyUsers[通知相关用户]
    notifyUsers --> displayChoice{用户角色}
    
    displayChoice -->|教师| displayTeacherReport[查看绩效报告]
    displayChoice -->|学生| displayStudentReport[查看个人成绩分析]
    displayChoice -->|管理员| displayAdminReport[查看综合统计报表]
    
    studentView --> displayStudentReport
    
    validationError --> endNode([结束])
    generateErrorReport --> endNode
    displayTeacherReport --> endNode
    displayStudentReport --> endNode
    displayAdminReport --> endNode

```

## 五、层级交互协议图

```mermaid
sequenceDiagram
    participant Client as 前端客户端<br/>(Vue 3)
    participant API as API网关<br/>(Flask)
    participant Service as 业务服务<br/>(Services)
    participant Analyzer as 分析引擎<br/>(GradeAnalyzer)
    participant DA as 数据访问层<br/>(Repository)
    participant DB as SQLite数据库
    
    Client->>API: HTTP POST /api/grades (JSON)
    API->>Service: gradeService.create(gradeDTO)
    Service->>DA: gradeDataAccess.insert(grade)
    DA->>DB: INSERT INTO student_grades
    DB-->>DA: 返回插入结果
    DA-->>Service: 返回成功
    Service->>Analyzer: triggerAnalysis(grade)
    Analyzer->>DA: queryTeacherCourses(subject)
    DA->>DB: SELECT * FROM teacher_courses
    DB-->>DA: 返回教师授课记录
    DA-->>Analyzer: 返回记录列表
    alt 记录存在
        Analyzer->>Analyzer: calculatePerformance()
        Analyzer->>DA: saveAnalysisResult(result)
        DA->>DB: INSERT INTO analysis_results
        DB-->>DA: 返回成功
        Analyzer-->>Service: 返回分析报告
        Service-->>API: 返回分析结果
        API-->>Client: 200 OK { report }
    else 记录不存在
        Analyzer-->>Service: 抛出异常(无授课记录)
        Service-->>API: 返回错误信息
        API-->>Client: 400 Error { message }
    end
```
