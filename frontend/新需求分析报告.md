# 新需求分析报告

## 需求概述
用户提出了6个具体需求，涉及三个成绩分析页面的功能增强：
1. 无法单独选择具体科目
2. 缺乏对具体科目的分析
3. 缺乏历次考试趋势分析
4. 没有具体科目的平均成绩，以及中位数
5. 课程表安排与学生成绩的关系分析
6. 任教教师对学生成绩的影响分析

## 工作内容分析

### 1. 前端功能增强

#### 页面组件修改
- **个人成绩分析页面**：
  - 添加科目选择下拉菜单
  - 新增具体科目分析模块
  - 新增历次考试趋势图表
  - 新增中位数统计展示

- **班级成绩分析页面**：
  - 添加科目选择下拉菜单
  - 新增具体科目分析模块
  - 新增班级历次考试趋势图表
  - 新增中位数统计展示

- **年级成绩分析页面**：
  - 添加科目选择下拉菜单
  - 新增具体科目分析模块
  - 新增年级历次考试趋势图表
  - 新增中位数统计展示
  - 新增教师成绩对比模块

#### 图表优化
- 新增折线图展示历次考试趋势
- 优化柱状图展示科目对比
- 新增箱线图展示成绩分布（包含中位数）
- 新增散点图展示课程安排与成绩关系
- 新增热力图展示教师与成绩关系

### 2. 后端API支持

#### 新增API端点
- **科目分析API**：
  - `/api/grades/analysis/subject/{studentId}/{subject}` - 个人科目分析
  - `/api/grades/analysis/class/{className}/{subject}` - 班级科目分析
  - `/api/grades/analysis/grade/{gradeName}/{subject}` - 年级科目分析

- **考试趋势API**：
  - `/api/grades/analysis/trend/{studentId}` - 个人考试趋势
  - `/api/grades/analysis/class/trend/{className}` - 班级考试趋势
  - `/api/grades/analysis/grade/trend/{gradeName}` - 年级考试趋势

- **教师分析API**：
  - `/api/grades/analysis/teacher/{subject}` - 教师成绩对比

- **课程安排分析API**：
  - `/api/grades/analysis/schedule/{studentId}` - 个人课程安排与成绩关系
  - `/api/grades/analysis/class/schedule/{className}` - 班级课程安排与成绩关系

#### 数据处理逻辑
- 计算中位数、标准差等统计指标
- 处理历次考试数据的时间序列分析
- 关联课程表数据与成绩数据
- 关联教师数据与班级成绩数据

### 3. 数据结构设计

#### 前端数据结构
- 扩展现有分析数据结构，添加科目详情字段
- 新增考试趋势数据结构

- 新增教师与成绩关系数据结构

#### 交互流程优化
- 科目选择联动机制

#### 响应式设计

### 5. 技术实现考虑

- Vue 3 Composition API

- SQLAlchemy ORM
- 统计分析库（如numpy、pandas）
- 缓存机制优化查询性能
