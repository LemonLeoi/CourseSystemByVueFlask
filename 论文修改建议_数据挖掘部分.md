### 1. 决策树模型实现

本系统基于决策树算法实现了学业状态预测功能，用于分析学生成绩的影响因素并进行学业预警。决策树模型具有结构清晰、易于理解的特点，适合应用于教育数据分析场景。系统实现了ID3和C4.5两种经典决策树算法，用户可根据数据特点灵活选择。

#### （1）数据预处理与特征工程

决策树算法要求输入数据为离散型特征，因此系统首先对原始成绩数据进行离散化处理。通过`preprocess_data`方法，将百分制成绩映射为五个等级：

| 分数区间 | 等级标签 |
|---------|---------|
| 90-100分 | 优秀 |
| 80-89分 | 良好 |
| 70-79分 | 中等 |
| 60-69分 | 及格 |
| 0-59分 | 不及格 |

离散化处理的代码实现如下：

```python
def preprocess_data(data):
    attr_names = ['学号', '性别', '班级', '年级', '考试类型', '学科', '分数范围']
    processed_data = []
    for row in data:
        score = row[6]
        if score >= 90:
            score_range = '优秀'
        elif score >= 80:
            score_range = '良好'
        elif score >= 70:
            score_range = '中等'
        elif score >= 60:
            score_range = '及格'
        else:
            score_range = '不及格'
        processed_row = row[:6] + [score_range]
        processed_data.append(processed_row)
    return processed_data, attr_names
```

#### （2）信息增益计算

决策树的核心是通过计算信息增益（或信息增益率）来选择最优划分属性。系统实现了`info_gain`函数，用于计算特征A对数据集D的信息增益。

**信息熵计算公式：**

$$H(D) = -\sum_{i=1}^{m}p_i\log_2 p_i$$

其中，$p_i$表示第i类样本在数据集D中所占的比例。

**信息增益计算公式：**

$$Gain(D, A) = H(D) - \sum_{v=1}^{V}\frac{|D^v|}{|D|}H(D^v)$$

其中，$D^v$表示特征A取值为v的样本子集。

**信息增益率计算公式（C4.5算法）：**

$$Gain\_ratio(D, A) = \frac{Gain(D, A)}{IV(A)}$$

其中，$IV(A) = -\sum_{v=1}^{V}\frac{|D^v|}{|D|}\log_2\frac{|D^v|}{|D|}$为特征A的固有值。

信息增益计算的代码实现如下：

```python
def info_gain(D, A, T=-1, return_ratio=False):
    # 统计各类别和属性值的数量
    C = {}   # 结果计数字典
    DA = {}  # 属性A的取值计数字典
    CDA = {} # 结果和属性A的组合计数字典
    
    for t in D:
        C[t[T]] = C.get(t[T], 0) + 1
        DA[t[A]] = DA.get(t[A], 0) + 1
        CDA[(t[T], t[A])] = CDA.get((t[T], t[A]), 0) + 1
    
    # 计算经验熵
    PC = map(lambda x: x / len(D), C.values())
    entropy_D = entropy(tuple(PC))
    
    # 计算经验条件熵
    PCDA = {}
    for key, value in CDA.items():
        a = key[1]
        pca = value / DA[a]
        PCDA.setdefault(a, []).append(pca)
    
    condition_entropy = 0.0
    for a, v in DA.items():
        p = v / len(D)
        e = entropy(PCDA[a])
        condition_entropy += e * p
    
    info_gain_value = entropy_D - condition_entropy
    
    if return_ratio:  # C4.5算法
        gain_ratio = info_gain_value / entropy_D if entropy_D > 0 else 0
        return {'gain': gain_ratio, 'info_gain': info_gain_value, 
                'entropy_before': entropy_D, 'entropy_after': condition_entropy}
    else:  # ID3算法
        return {'gain': info_gain_value, 'info_gain': info_gain_value,
                'entropy_before': entropy_D, 'entropy_after': condition_entropy}
```

#### （3）决策树构建算法

系统通过递归方式构建决策树，核心函数为`build_tree`。算法流程如下：

1. **终止条件判断**：若所有样本属于同一类别，或特征集为空，或达到最大深度，则创建叶子节点
2. **特征选择**：计算所有特征的信息增益（或信息增益率），选择最优特征作为分裂属性
3. **数据集划分**：根据最优特征的取值将数据集划分为多个子集
4. **递归构建子树**：对每个子集递归调用构建算法

决策树构建支持以下参数配置：

| 参数名 | 默认值 | 说明 |
|-------|-------|------|
| algorithm | C4.5 | 算法类型（ID3或C4.5） |
| max_depth | 10 | 最大树深度 |
| min_samples_split | 2 | 最小分裂样本数 |
| threshold | 0.0001 | 信息增益阈值 |

#### （4）模型输出与可视化

决策树构建完成后，系统生成树结构数据，包含每个节点的以下信息：

- 节点ID和深度
- 分裂属性及属性名称
- 信息增益和信息增益率
- 分裂前后的熵值
- 样本数量和类别分布
- 子节点信息

系统通过`visualize_tree`函数将决策树转换为可视化数据格式，前端使用ECharts进行树形图渲染，直观展示决策路径和分类规则。

#### （5）学业预警应用

训练完成的决策树模型可用于学业预警。系统将学生的历史成绩特征输入模型，沿决策树路径进行判断，最终到达叶子节点获得分类结果。通过分析决策路径，教师可以了解影响学生学业表现的关键因素，从而制定针对性的教学干预措施。
