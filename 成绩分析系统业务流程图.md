# 学校管理系统 - 成绩分析系统业务流程图

## 一、成绩分析完整生命周期流程

```mermaid
flowchart TD
    subgraph Input["一、数据输入层 Data Input"]
        direction TB
        start([开始]) --> roleSelect{用户角色选择}

        roleSelect -->|教师| teacherLogin[教师登录系统]

        roleSelect -->|班主任| classTeacherLogin[班主任登录系统]

        teacherLogin --> selectAction{选择操作}
        adminLogin --> selectAction
        studentLogin --> selectAction
        classTeacherLogin --> selectAction

        selectAction -->|录入成绩| gradeInput[输入成绩数据]
        selectAction -->|查看分析| viewAnalysis[查看成绩分析]

        validateScore -->|检查科目| validateSubject{科目有效性?}
        validateSubject -->|检查考试| validateExam{考试存在性?}

        validateStudentID -->|无效| errorSID[错误: 学号不存在]
        validateScore -->|无效| errorScore[错误: 成绩超出范围]
        validateSubject -->|无效| errorSubject[错误: 科目不存在]

    subgraph Storage["三、数据存储层 Data Storage"]

        validateExam -->|有效| saveGrade[保存成绩到数据库]

        storeProcess -->|写入成绩表| insertGrade["INSERT INTO student_grades"]
        storeProcess -->|更新考试统计| updateExamStats["UPDATE exam_statistics"]
        storeProcess -->|触发分析| triggerAnalysis[触发自动分析]

        triggerAnalysis -->|查询学生| queryStudent[查询学生信息]

        preprocess -->|数据清洗| dataCleaning[清洗异常数据]
        dataCleaning -->|缺失值处理| handleMissing[处理缺失成绩]
        handleMissing -->|异常值检测| detectOutliers[检测离群点]
        detectOutliers -->|数据转换| dataTransform

        preprocess -->|数据转换| dataTransform[格式化数据]
        dataTransform -->|统一编码| normalizeCode[标准化编码]
        normalizeCode -->|数据整合| dataIntegration

        preprocess -->|数据整合| dataIntegration[整合多源数据]
        dataIntegration -->|关联学生| linkStudent[关联学生信息]
        dataIntegration -->|关联课程| linkCourse[关联课程信息]
        dataIntegration -->|关联教师| linkTeacher[关联教师信息]

    subgraph Analysis["五、数据分析层 Data Analysis"]
        direction TB

        selectAnalysisType -->|个人分析| personalAnalysis[个人成绩分析]
        selectAnalysisType -->|班级分析| classAnalysis[班级成绩分析]
        selectAnalysisType -->|年级分析| gradeLevelAnalysis[年级成绩分析]
        selectAnalysisType -->|科目分析| subjectAnalysis[科目成绩分析]
        selectAnalysisType -->|趋势分析| trendAnalysis[考试趋势分析]
        selectAnalysisType -->|教师绩效| teacherAnalysis[教师绩效对比]

        personalAnalysis --> personalStats[计算个人统计指标]
        classAnalysis --> classStats[计算班级统计指标]
        gradeLevelAnalysis --> gradeStats[计算年级统计指标]
        subjectAnalysis --> subjectStats[计算科目统计指标]
        trendAnalysis --> trendStats[计算趋势统计指标]
        teacherAnalysis --> teacherStats[计算教师绩效指标]

        personalStats --> calcMetrics[计算分析指标]
        classStats --> calcMetrics
        gradeStats --> calcMetrics
        subjectStats --> calcMetrics
        trendStats --> calcMetrics
        teacherStats --> calcMetrics

        calcMetrics -->|计算平均值| calcAverage["平均值<br/>Mean"]
        calcMetrics -->|计算标准差| calcStdDev["标准差<br/>Std Dev"]
        calcMetrics -->|计算中位数| calcMedian["中位数<br/>Median"]
        calcMetrics -->|计算分布| calcDistribution["成绩分布<br/>Distribution"]
        calcMetrics -->|计算排名| calcRank["班级/年级排名"]

        calcAverage --> compareToClass[与班级平均对比]
        calcStdDev --> analyzeStability[分析成绩稳定性]
        calcMedian --> analyzeMedianDiff[分析中位数差异]
        calcDistribution --> analyzeDistShape[分析分布形态]
        calcRank --> generateRankings[生成排名列表]

        compareToClass --> identifyStrengths[识别优势学科]
        compareToClass --> identifyWeaknesses[识别劣势学科]
        analyzeStability --> performanceStability[评估稳定性]

        chartDisplay -->|雷达图| radarChart[学科能力雷达]

        barChart --> displayChart[展示图表]
        lineChart --> displayChart
        pieChart --> displayChart
        radarChart --> displayChart
```

## 二、个人成绩分析详细流程

```mermaid
flowchart TB
    subgraph PersonalAnalysis["个人成绩分析流程"]
        startPA([开始个人分析]) --> inputStudentID[输入学生学号]

        inputStudentID --> validateStudent{学生存在性验证}
        validateStudent -->|存在| queryStudentInfo[查询学生基本信息]
        validateStudent -->|不存在| returnError1[返回错误: 学生不存在]

        queryStudentInfo --> queryGrades[查询该学生所有成绩]
        queryGrades --> checkGrades{是否有成绩?}

        checkGrades -->|无成绩| returnNoGrade[返回: 暂无成绩记录]
        checkGrades -->|有成绩| continueAnalysis

        continueAnalysis --> organizeByExam[按考试组织成绩]
        organizeByExam --> organizeBySubject[按科目整理]

        organizeBySubject --> calculateSubjectAvg[计算各科目平均分]
        calculateSubjectAvg --> calculateOverallAvg[计算整体平均分]

        calculateOverallAvg --> queryClassAvg[查询班级平均成绩]
        queryClassAvg --> compareWithClass[与班级平均对比]

        compareWithClass --> identifyStrong{优势学科?}
        identifyStrong -->|成绩>班级平均| addToStrengths[标记为优势学科]

        identifyStrong --> identifyWeak{劣势学科?}
        identifyWeak -->|成绩<班级平均| addToWeaknesses[标记为劣势学科]

        addToStrengths --> sortByDiff[按差异排序]
        addToWeaknesses --> sortByDiff

        sortByDiff --> top3Strengths[选取前3优势学科]
        sortByDiff --> top3Weaknesses[选取前3劣势学科]

        top3Strengths --> generateEval[生成综合评价]
        top3Weaknesses --> generateEval

        generateEval -->|整体优秀| eval1["评价: 优秀"]
        generateEval -->|高于平均| eval2["评价: 良好"]
        generateEval -->|接近平均| eval3["评价: 中等"]
        generateEval -->|低于平均| eval4["评价: 需提高"]

        eval1 --> buildResult[构建分析结果]
        eval2 --> buildResult
        eval3 --> buildResult
        eval4 --> buildResult

        buildResult -->|返回JSON| endPA([返回分析结果])
        returnError1 --> endPA
        returnNoGrade --> endPA
    end

    style startPA fill:#E3F2FD,stroke:#1565C0
    style endPA fill:#E8F5E9,stroke:#2E7D32
    style returnError1 fill:#FFEBEE,stroke:#C62828
    style returnNoGrade fill:#FFF3E0,stroke:#E65100
```

## 三、班级成绩分析详细流程

```mermaid
flowchart TB
    subgraph ClassAnalysis["班级成绩分析流程"]
        startCA([开始班级分析]) --> inputClassInfo[输入班级信息]

        inputClassInfo --> validateClass{班级存在性验证}
        validateClass -->|存在| queryClassStudents[查询班级学生列表]
        validateClass -->|不存在| returnError2[返回错误: 班级不存在]

        queryClassStudents --> countStudents{学生数量>0?}
        countStudents -->|是| continueCA
        countStudents -->|否| returnNoStudents[返回: 班级无学生]

        continueCA --> queryAllGrades[查询班级所有成绩]
        queryAllGrades --> checkClassGrades{是否有成绩?}

        checkClassGrades -->|无成绩| returnNoClassGrade[返回: 暂无成绩]
        checkClassGrades -->|有成绩| organizeByStudent

        organizeByStudent --> organizeByExam[按考试组织]
        organizeByExam --> organizeBySubjectCA[按科目整理]

        organizeBySubjectCA --> calculateSubjectAvgCA[计算各科目班级平均]
        calculateSubjectAvgCA --> calculateClassOverall[计算班级整体平均]

        calculateClassOverall --> calculateClassStats[计算班级统计数据]
        calculateClassStats -->|最高分| findMax[找出最高分]
        calculateClassStats -->|最低分| findMin[找出最低分]
        calculateClassStats -->|平均分| calcClassAvg[重新计算平均]

        findMax --> generateClassReport[生成班级分析报告]
        findMin --> generateClassReport
        calcClassAvg --> generateClassReport

        generateClassReport -->|学科对比| subjectCompare[班级各学科对比]
        generateClassReport -->|学生排名| studentRanking[学生成绩排名]
        generateClassReport -->|考试分析| examAnalysis[历次考试分析]

        subjectCompare -->|超过年级平均| classStrength[班级优势学科]
        subjectCompare -->|低于年级平均| classWeakness[班级劣势学科]

        studentRanking -->|前10%| topStudents[优秀学生名单]
        studentRanking -->|后10%| bottomStudents[需要关注学生]

        examAnalysis -->|成绩上升| improveTrend[进步趋势]
        examAnalysis -->|成绩下降| declineTrend[退步趋势]
        examAnalysis -->|成绩稳定| stableTrend[稳定状态]

        classStrength --> buildClassResult[构建班级分析结果]
        classWeakness --> buildClassResult
        topStudents --> buildClassResult
        bottomStudents --> buildClassResult
        improveTrend --> buildClassResult
        declineTrend --> buildClassResult
        stableTrend --> buildClassResult

        buildClassResult --> endCA([返回班级分析结果])
        returnError2 --> endCA
        returnNoStudents --> endCA
        returnNoClassGrade --> endCA
    end

    style startCA fill:#E8F5E9,stroke:#2E7D32
    style endCA fill:#E1F5FE,stroke:#0277BD
    style returnError2 fill:#FFEBEE,stroke:#C62828
```

## 四、统计分析算法流程

```mermaid
flowchart TB
    subgraph Statistics["统计分析算法流程"]
        subgraph Input["统计输入"]
            scoresInput([输入成绩列表]) --> validateScores
            validateScores -->|非空| continueStats
            validateScores -->|为空| returnZero[返回零值]
        end

        continueStats -->|计算均值| calcMean
        subgraph MeanCal["平均值计算"]
            calcMean --> sumScores["求和 Σx"]
            sumScores --> countScores["计数 n"]
            sumScores --> divideSum["sum / n"]
            divideSum --> meanResult["结果: x̄"]
        end

        meanResult -->|计算标准差| calcStd
        subgraph StdCal["标准差计算"]
            calcStd --> diffFromMean["(x - x̄)²"]
            diffFromMean --> sumSquared["求和"]
            sumSquared --> divideByN["sum / (n-1)"]
            divideByN --> sqrtVariance["√方差"]
            sqrtVariance --> stdResult["结果: s"]
        end

        stdResult -->|计算中位数| calcMed
        subgraph MedianCal["中位数计算"]
            calcMed --> sortScores["排序"]
            sortScores --> findMiddle{"奇数/偶数?"}
            findMiddle -->|奇数| middleOdd["中间值"]
            findMiddle -->|偶数|"中间两值平均"
            middleOdd --> medianResult["结果: Md"]
        end

        medianResult -->|计算分布| calcDist
        subgraph DistCal["分布统计"]
            calcDist --> countExcellence[计数90-100分]
            calcDist --> countGood[计数80-89分]
            calcDist --> countAverage[计数70-79分]
            calcDist --> countPass[计数60-69分]
            calcDist --> countFail[计数0-59分]

            countExcellence --> distResult["分布结果"]
            countGood --> distResult
            countAverage --> distResult
            countPass --> distResult
            countFail --> distResult
        end

        distResult -->|计算百分位| calcPercentile
        subgraph PercentileCal["百分位计算"]
            calcPercentile --> sortForPercentile["排序"]
            sortForPercentile --> calcPosition["位置 = P/100 × (n+1)"]
            calcPosition --> findValue["确定百分位值"]
            findValue --> percentileResult["百分位结果"]
        end

        percentileResult --> outputStats([输出统计结果])

        returnZero --> outputStats
    end

    subgraph Formulas["计算公式"]
        formula1["x̄ = Σx / n"]
        formula2["s = √[Σ(x-x̄)²/(n-1)]"]
        formula3["Md = 中间值"]
        formula4["P = (k/n) × 100%"]
    end

    style Input fill:#E3F2FD
    style MeanCal fill:#E8F5E9
    style StdCal fill:#FFF3E0
    style MedianCal fill:#FCE4EC
    style DistCal fill:#F3E5F5
    style PercentileCal fill:#E1F5FE
    style Formulas fill:#ECEFF1
```
